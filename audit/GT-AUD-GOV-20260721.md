## §D Prioritized change plan

### Phase 1 — Stop drift (High, low risk)

**D1. Slim `00-core-pipeline.md` to pointer**  
- *File:* `.agents/rules/00-core-pipeline.md`  
- *Change:* Replace pipeline list with: "On website brief → execute pipeline in `@AGENTS.md`." Keep brief-detect + do-not-stop-early.  
- *Effect:* Single pipeline source; ~600 chars freed.

**D2. Slim `AGENTS.md` hub/off-limits to `@` rules**  
- *File:* `AGENTS.md`  
- *Change:* Hub contract → `@01-site-v2-standard.md`; off-limits git → `@03-repo-safety.md`. Keep pipeline + commands table (operational).  
- *Effect:* `AGENTS.md` = procedure index, not schema encyclopedia.

**D3. Replace brief §2 verbatim block with `@` contract**  
- *File:* `.agents/prompts/daily-brief-generator.md`  
- *Change:* §2 outputs: "Follow `@AGENTS.md`, `@01-site-v2-standard.md`, `@acquire-images`, `@qa-and-ship`. Slug: `sites/<slug>/`." Remove 15-line paste block.  
- *Effect:* Briefs shrink; builder invariants don't re-copy daily.

**D4. Fix skill typos**  
- *Files:* `acquire-images/SKILL.md` L27, `optimize-assets.md` L16  
- *Change:* `.webp`/`.webp` → `.webp` handoff wording.  
- *Effect:* Clearer WebP gate language.

### Phase 2 — Enforce pipeline goals (Critical/High)

**D5. Codebase-aware brief — builder-side scan (required regardless of Gemini access)**  
- *Files:* `.agents/skills/parse-brief/SKILL.md`, `.agents/skills/research-and-plan/SKILL.md`  
- *Change:* Mandatory step 3: read `sites/*/meta.json` → emit slug availability, existing titles, layout families used (from brief §3 if retrievable). Block silent overwrite; flag near-duplicate brands.  
- *Effect:* Enforces collision/similarity at build even when Gemini is blind.

**D6. Codebase-aware brief — producer-side index**  
- *Files:* new `scripts/export-sites-index.js` → `.agents/prompts/_sites-index.md` (generated artifact); `daily-brief-generator.md` STEP 0  
- *Change:* Add: "Before niche pick, consult `_sites-index.md` if provided; reject slug collisions; avoid layout family used in last 7 slugs." Scheduled Action operator pastes/attaches index weekly or per deploy.  
- *Effect:* Gemini can avoid collisions without repo filesystem access.

**D7. Content depth enforcement script**  
- *Files:* new `scripts/check-copy-depth.js`; `package.json` (out of scope but referenced); `.agents/skills/qa-and-ship/SKILL.md`  
- *Change:* QA step: script counts visible text vs brief §3 floor (parsed from checklist). Fail QA if below floor −10%.  
- *Effect:* Thin copy becomes a **hard gate**, not prose.

**D8. WebP gate in QA script**  
- *Files:* `qa_sweep.js` (app source — note in plan); `.agents/skills/qa-and-ship/SKILL.md`  
- *Change:* Add check: photographic `<img src>` must end `.webp`. Fail report entry.  
- *Effect:* WebP invariant machine-enforced.

**D9. Strengthen `design-and-build` copy gate**  
- *File:* `.agents/skills/design-and-build/SKILL.md`  
- *Change:* Before handoff: self-check word counts per §4b min/max; require ≥N directed sections per architecture type from §3.  
- *Effect:* Builder cannot skip §4b modules.

### Phase 3 — Activation hygiene & structure (Medium)

**D10. Retarget `01-site-v2-standard` activation**  
- *File:* `.agents/rules/01-site-v2-standard.md`  
- *Change:* `trigger: glob` → `sites/**` (or model_decision + glob description).  
- *Effect:* Saves AlwaysOn context on hub-only turns.

**D11. Deliberate producer/consumer split for top-level docs**  
- *Files:* `GEMINI.md`, `AGENTS.md`  
- *Change:* See §F — implement model (b). `GEMINI.md` = stack + Gemini ops + `@AGENTS.md` for builder. `AGENTS.md` = builder SOT.  
- *Effect:* Clear ownership; reduced overlap.

**D12. Add `.gemini/settings.json`**  
- *File:* `.gemini/settings.json` (new)  
- *Change:* `{ "context": { "fileName": "GEMINI.md" } }`  
- *Effect:* Gemini project auto-loads producer context.

### Phase 4 — Optional hardening (Medium/Low)

**D13. `hooks.json` for deterministic gates**  
- *File:* `.agents/hooks.json` or repo-root `hooks.json` (per Antigravity convention — confirm path)  
- *Change:* `PostToolUse` on Write to `sites/**/meta.json`: reject `"qa":"v2-pass"` if `qa-report.json` lacks PASS for slug. `PostToolUse` on Write to `sites/**/*.html`: warn on `.webp`/`.webp` in `<img src>`.  
- *Effect:* Catches agent mistakes before ship.

**D14. `plugin.json` bundle**  
- *File:* `.agents/plugin.json` (new)  
- *Change:* Package rules, skills, workflows, `mcp_config.json` for portable Antigravity install.  
- *Effect:* Single install unit; versioned governance layer.

**D15. Document empty MCP**  
- *File:* `.agents/mcp_config.json`  
- *Change:* Add `"_comment": "Intentionally empty; image gen uses built-in tools per README."` OR JSONC note in README only.  
- *Effect:* Resolves F11 without adding servers.

**D16. Cursor documentation**  
- *File:* `GEMINI.md` or README  
- *Change:* Note Cursor as alternate builder IDE using same `AGENTS.md` + `.agents/skills/`.  
- *Effect:* Consumer map complete.
